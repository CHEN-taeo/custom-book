# Custom Book — 竞品分析

## 竞争格局

| 产品 | 形态 | 个性化 | 知识溯源 | 长篇叙事 | Custom Book 机会 |
|------|------|--------|----------|----------|------------------|
| [NotebookLM](https://notebooklm.google/) | 文档 QA + 播客 + 闪卡 | 中 | **强** | 弱 | 可读可追更的小说，非播客 |
| [Lorey](https://www.loreyai.com/) | 笔记 → 宇宙剧集 ~8min | **强** | 中 | 弱 | 长篇 + 掌握度 + 知识对照 |
| [StudyRPG](https://studyrpg.ai/) | 笔记 → Pokémon 式 RPG | 中 | 中 | 无 | 吸引爱读小说不爱玩游戏用户 |
| [StudyQuest](https://studyquest.app/) | 15+ 小游戏 | 中 | 弱 | 无 | 叙事沉浸 vs 街机游戏 |
| [Langory](https://langory.com/) | K12 互动故事 + 自适应 | **强** | 中 | 中 | 成人/自学者 + 任意材料 |
| [ZunoTales](https://www.zunotales.com/) | 课标故事 + 插图 | 中 | 中 | 短 | 非 K12，自由题材 |
| [ImmersiveBook](https://www.immersivebook.app/) | 文档 → 分支互动 | 中 | 中 | 中 | 小说追更感 + SRS |

## 竞品启示

### NotebookLM
- Grounding in sources 是信任基础
- 多模态（音频、闪卡、quiz）提高留存
- Learning Guide 式苏格拉底提问 → 可做成「角色导师」对话

### Lorey
- 「选 universe」是极强的 onboarding 钩子
- 流媒体式 UI（episodes、cliffhanger）比 dashboard 更粘
- 订阅制按「故事数」计费已被验证

### Langory / ZunoTales
- 自适应难度、课标标签是 B2B 卖点
- 学校市场需要 roster、合规、分析报表

### StudyRPG / StudyQuest
- 游戏化留存高，但受众偏年轻/偏游戏
- Custom Book 坚持 **阅读体验** 而非操作体验

## SWOT

| | |
|---|---|
| **S** | 差异化清晰：知识 × 长篇定制小说 × 掌握度闭环 |
| **W** | 长文生成成本高；幻觉风险；生成慢 |
| **O** | Lorey 验证品类；NotebookLM 教育化用户；中文自学者市场大 |
| **T** | 大厂随时做 Story Mode；同质化 AI 写作工具多 |

## 定位语句

> **NotebookLM 帮你听懂，Custom Book 帮你读进去。**

> **把任何教材，变成一本只为你写的小说。**
