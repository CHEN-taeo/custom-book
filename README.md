# Custom Book

> 把要学习的知识点，量身定制成一本你喜欢的小说。

**Custom Book** 是一款 AI 驱动的学习阅读应用：上传任意学习材料，选择你喜欢的小说类型与主角设定，系统将知识点结构化后嵌入连载叙事，让你在「追更」中完成理解与记忆。

---

## 核心洞察

人们不爱读「知识」，但爱读「故事」。同一套知识点，可以换不同的叙事外壳——武侠里的暗号接头、赛博朋克里的通讯握手、悬疑里的线索拼图——**知识点不变，体验随人定制**。

## 产品价值

| 传统方式 | Custom Book |
|---------|-------------|
| 教材枯燥，难以坚持 | 熟悉的小说框架降低认知门槛 |
| 读完不等于学会 | 章末测验 + 概念掌握度追踪 |
| 一种写法适合所有人 | 同一知识可切换题材（修仙版 / 悬疑版 / 科幻版） |
| 复习靠意志力 | 薄弱概念自动「剧情回流」与间隔复习 |

## 用户旅程

```
上传材料 → AI 解析知识点 → 确认大纲 → 选择小说偏好
    → 生成 Book Bible → 连载阅读 → 章末测验 → 自适应下一章
```

## 差异化定位

不是「AI 写小说玩具」，也不是「再做一套 flashcard」：

- **叙事外壳可变** — 题材、文风、主角深度定制
- **知识内核可追溯** — 每段故事绑定知识锚点，可对照原文
- **学习闭环** — 掌握度驱动情节走向与复习插入

> NotebookLM 帮你听懂，Custom Book 帮你读进去。

## MVP 范围

- [ ] 文本 / PDF 导入与知识提取
- [ ] 题材与主角偏好设定
- [ ] Book Bible + 章节计划生成
- [ ] 流式章节阅读器
- [ ] 章末测验与概念掌握度
- [ ] 知识对照模式（故事段落 ↔ 原文）

## 技术栈（规划）

- **前端**: Next.js + React + Tailwind
- **后端**: Next.js API / FastAPI
- **数据**: PostgreSQL + pgvector
- **AI**: LLM 多阶段 Agent 流水线 + RAG
